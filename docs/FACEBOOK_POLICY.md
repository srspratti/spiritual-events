# Facebook Event Policy

This product does not scrape Facebook.

Facebook events may be added only through one of these paths:

1. Manual organizer submission of a public event link for admin review.
2. Meta Graph API access after required Meta app review and organizer/admin authorization.
3. Partner organizer exports or calendar feeds where sharing is permitted.

The app must not import:

- Private Facebook groups
- Private Facebook events
- Member-only content
- Group member lists
- Event attendee lists
- Personal user profile data
- Any data collected through unauthorized automation

Suggested user-facing note:

> Facebook events are shown only when the event, Page, Group, or organizer has granted the required permissions or when an organizer submits the event for review. We do not access private Facebook groups, private events, member-only content, or personal Facebook data unless permitted by the platform and the relevant authorization. Some Facebook events may not appear in the app even if you are personally a member of the group.
